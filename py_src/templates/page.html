ABOUT THE APP

We have created a desktop app that you can log into that takes your login information and system specs to make a smart contract and put onto a block on the blockchain to check in the future if you are using the same computer or authenticate on a new device.

The desktop app that we designed uses solidity to make a smart contract that interacts with the login system. We do this to make a very secure system that uses both username/password combined with sytem specs to make a very secure login process. The use of system specs for credentials is to stop malicious actors from getting into your account using other computers so unless you swap parts in your build you should only need to authenticate once on your computer. This keeps things much more secure than if you just had a username and password alone with no system specs that are being cross referenced.
